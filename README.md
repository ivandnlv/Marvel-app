# Веб-приложение по работе с Marvel Api

В данном проекте я познакомился с RTK Query, также закрепил на нем Redux Toolkit. 

Запуск:
```
npm install
```
```
npm start
```

Стэк: 
+ React hooks;
+ Redux toolkit;
+ RTK Query;
+ Scss modules;
